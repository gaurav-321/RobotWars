Installation

For Windows
⬇️  Install Requirements.txt by using command:-
        pip install -r requirements.txt
        Run the executable named main_file.py inside the extracted file.


Linux/Debian based systems
    Install Requirements.txt by using command:-
        pip3 install -r requirements.txt
        Run the executable named main_file.py inside the extracted file.



Controls:-


Press left click to see the next move.
Set deactivated robots from menu
Game will rest if the robots are destroyed.
Destroyed deactivated robots drop guns and body
Picking a body restores the hp
Heath bar is shown at the top
